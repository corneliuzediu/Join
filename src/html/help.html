What is Join?

Join is a business oriented tool that focusses on helping a team (or an individual) to keep track of a Business Project. The application summarizes the state of all tasks on a "Summary" dashboard.

The "Board" shows all tasks in more detail and sorts them by their status. When an additional task needs to be added for the team, the Team-Leader has the option to create a new task via the "Add Task" section or directly within the "Board".

The section "Contacts" lets the user save individuall contacts and their details.

How to use it

    1. After logging in the user is being welcomed on the summary dashboard by showing all the tasks registered. A click on any element directs the user to the "Board" section.
    2. The "Board" section provides all the tasks in their current state and with the following options:
        - Drag & Drop: each task to a different progress status lane
        - View: the details of a task by clicking on it
        - Edit: the task by clicking on the edit icon in the view card. The user can now modify all details of the task
        - Add: a new task by clicking the "+" icon in the board or "Add Task" in the sidebar
        - Filter: all tasks with the search option. All matching tasks will simultaneously be displayed as a result
    3. The "Contacts" section allows to store name, email and phone number of each team member. All contacts are available via multiple choice when opening the "Add-task" dialog. For your convenience, you can directly create a new task on the contact detail card and you can choose to edit the details of the contact here.
