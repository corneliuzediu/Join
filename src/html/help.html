Help

What is Join?

Join is a bussiness oriented tool that is focused on helping a team (or an individual) to keep track of the Bussiness Projects. The application offers the posibility of a summarised overview throw "Summary" section, followed up by a more detailed area, such as "Board" section.
When required to insert additional tasks for the team, the Team-Leader have the possibility to create it via "Add Task" section or direct on "Board section". Additionally, the section "Contact" provides the possibility of saving individually the members contacts data.

How to use it

    1. After loging in, the user is being welcome by an overview of all the tasks registered. The application can provide more informations by selecting any of the elements from the "Summary" section or by going on "Board" section.
    2. The "Board" section provides all the tasks in their current state and with the following availabilities:
        - Drag & Drop: in the requirenments of developing the task;
        - Edit: when needed, by selecting the task, the user have the posibility to modify it. (ex. due date, assigned memmbers, etc.)
        - Add: to provide a multiple acces point, the user have the option to add new task throw "Board" section.
        - Filter: a search option have been provided in order to find faster an specific task. The task will be displayed in the current state section.
    3. If required to add a new task, the user has also this option via "Add Task" section.
    4. In order to keep easy contact with the other members, the user has the possibility to save their contact data in "Contact" section.
